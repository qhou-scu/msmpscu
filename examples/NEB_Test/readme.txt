This folder conatins a number of input files for testing NEB simulations


to run the examples:
On linux:  
   run Commandline: SUBSTRATE.exe "Setup_test1.dat" 0 2
   run Commandline: SUBSTRATE.exe "Setup_test2.dat" 0 2
   run Commandline: SUBSTRATE.exe "Setup_test3.dat" 0 2
   
In VisualStudio(windows): 
   open the project SUBSTRATE
   open the property page of the poject
   set  "Working Directory" to be the present directory
   set  "Application Arguments" to be: "Setup_test1.dat" 0 2 
or set  "Application Arguments" to be: "Setup_test2.dat" 0 2 
   choice the menu: "Debug"-"Start Without Debugging"
