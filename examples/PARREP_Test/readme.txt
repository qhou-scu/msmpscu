This folder conatins a number of input files for testing parallel-replica MD

to run the example:
On linux:  
   run Commandline: SUBSTRATE.exe "SampleSetup.dat" 0 2
   
In VisualStudio(windows): 
   open the project SUBSTRATE
   open the property page of the poject
   set  "Working Directory" to be the present directory
   set  "Application Arguments" to be: "SampleSetup.dat" 0 2 
   choice the menu: "Debug"-"Start Without Debugging"
