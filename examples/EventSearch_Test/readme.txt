This folder conatins a number of input files for testing EventSearch
which is a extension of GMD. 
See also GMD_Test

to run the example:
On linux:  
   run Commandline: EventSearch_GPU.exe "SampleSetup.dat" 0 2
   
In VisualStudio(windows): 
   open the project EventSearch
   open the property page of the poject
   set  "Working Directory" to be the present directory
   set  "Application Arguments" to be: "SampleSetup.dat" 0 2 
   choice the menu: "Debug"-"Start Without Debugging"
